# Setup

1. Copy `.env.example` to `.env.local`.
2. Set `NEXT_PUBLIC_STORE_MODE=demo` for development.
3. Optionally add Stripe test keys and `CONTACT_EMAIL`.
4. Run `npm install`.
5. Run `npm run dev`.
6. Open `http://localhost:3000`.

Never commit secrets. Live mode requires the launch checklist in `docs/launch-checklist.md`.
